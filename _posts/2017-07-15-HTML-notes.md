# AJAX
AJAX = 异步 JavaScript 和 XML。<br />
AJAX 是一种用于创建快速动态网页的技术。<br />
通过在后台与服务器进行少量数据交换，AJAX 可以使网页实现异步更新。这意味着可以在不重新加载整个网页的情况下，对网页的某部分进行更新。<br />
传统的网页（不使用 AJAX）如果需要更新内容，必需重载整个网页面。
