# python 2 to 3

1. 打开windows的cmd，进入Python安装的根目录\Python27\Tools\Scripts。（2to3.py在此目录下）
2. python 2to3.py -w 要转换的文件.py<br />
【比如（python 2to3.py -w e:\pycode\perceptron.py）】<br />
生成bak文件是备份文件，可直接改后缀变成相应的python文件即可。
